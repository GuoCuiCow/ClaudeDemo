# 任务型 Skill 示例

这是一个任务型skill，用于演示Claude Code如何执行具体的文件处理任务。

## 功能说明

读取指定文件并生成结构化摘要，包含以下信息：
- 文件基本信息（路径、大小、行数）
- 内容结构分析
- 关键要点提取

## 使用方法

```bash
# 方式1: 通过Skill工具
Skill({ skill: "demo-task", args: "README.md" })

# 方式2: 手动触发后输入文件路径
/demo-task
```

## 执行步骤

1. **接收参数** - 获取用户提供的文件路径
2. **读取文件** - 使用Read工具读取文件内容
3. **分析内容** - 提取文件结构和关键信息
4. **生成摘要** - 输出结构化的分析结果

## 输出格式

```
## 文件摘要

### 基本信息
- 路径: {file_path}
- 大小: {size} bytes
- 行数: {line_count}

### 内容结构
{分析结果}

### 关键要点
{提取的关键信息}
```

## 示例输出

当处理README.md时，输出类似：

```
## 文件摘要

### 基本信息
- 路径: README.md
- 大小: 2887 bytes
- 行数: 143

### 内容结构
- 安装说明（多种平台）
- 基础用法（启动、命令、参数）
- 核心功能（文件操作、执行命令、记忆系统）
- 最佳实践（工作流、上下文管理）

### 关键要点
- 这是一个Claude Code快速入门指南
- 覆盖安装、配置、核心功能、最佳实践
- 包含MCP集成和技能系统介绍
```

## 错误处理

- 文件不存在: 提示用户检查路径
- 权限问题: 说明需要读取权限
- 空文件: 提示文件为空

> 提示: 任务型skill执行具体操作，返回可用的结果。
